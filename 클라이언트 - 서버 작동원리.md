# 클라이언트 - 서버 작동원리

```diff
- 클라이언트 - 서버 구조
+ 클라이언트는 어떤 역할인지, 서버는 어떤 역할인지에 대해서 자세히 찾아볼 것

- 데이터 전송
+ 웹은 일반적으로 JSON을 통하여 데이터를 주고받는데, JSON이란 무엇인지에 대해서 찾아볼 것
+ 클라이언트와 서버가 데이터를 주고 받기 위해서 어떤 것이 필요한지 찾아볼 것

- CORS 정책
+ CORS (Cross-Origin Resource Sharing)이란 무엇인지에 대해서 찾아볼 것

- REST API
+ REST API(Restful API)란 무엇인지, 일반적인 API와 어떤 차이점이 있는지에 대해서 찾아볼 것

- HTTP
+ HTTP(HyperText Trasfer Protocol)란 무엇인지, 프로토콜은 무엇인지에 대해서 찾아볼 것

- WAS
+ WAS(Web Application Server)란 무엇인지, 일반적인 웹 서버와 무엇이 다른지에 대해서 찾아볼 것
```
