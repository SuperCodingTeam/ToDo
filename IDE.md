# IDE

```diff
- IDE 개념
+ IDE(Integrated Development Environment)란 무엇인지 학습할 것

- 대표 IDE
+ 대표적으로 많이 사용하는 IDE 에 대해서 알아볼 것
+ Visual Studio Code, IntelliJ(대부분 Java 또는 Kotlin), Pycharm(대부분 Python) 등등 많음

- JetBrains
+ JetBrains는 IDE 개발을 중점으로 하는 개발 업체
+ JetBrains에 정말 많은 IDE가 있고, 대학생은 무료로 사용할 수 있으니 찾아볼 것

- Visual Studio Code
+ Visual Studio Code는 현재 가장 많이 사용되는 IDE중 하나
+ Visual Studio Code의 특별한 기능에 대해서 찾아볼 것 (.vscode, settings.json, tasks.json 등등)
```
