# git 과 github

<hr />

```diff
- git과 github란?
+ git과 github란 무엇인지 찾아보고, 둘은 어떤 차이점이 있는지 찾아볼 것

- git 명령어
+ git을 사용하기 위한 기본적인 명령어에 대해서 알아볼 것

- git 명령어 (WorkSpace -> Remote Repository)
+ git add, git commit, git push

- git 명령어 (Remote Repository -> WorkSpace)
+ git clone, git pull, git fetch

- branch
+ 협업을 하기 위한 github의 branch 개념에 대해서 알아볼 것

- Pull Request
+ 협업을 하기 위한 Fork 개념 및 PR(Pull Request)의 개념에 대해 알아볼 것

```
