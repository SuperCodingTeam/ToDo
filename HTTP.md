# HTTP

```diff
- HTTP란?
+ HTTP에 대해서 학습해 보고, 웹에서 어떻게 작동하는지 찾아볼 것

- HTTPS
+ HTTP와 HTTPS의 차이점에 대해 학습해 보고, HTTPS는 어떻게 사용할 수 있는지 찾아볼 것 (SSL/TLS 인증서)

- HTTP 메소드
+ HTTP 메소드중 GET, POST, PUT, PATCH, DELETE 메소드에 대해서 학습해 볼것

- HTTP 상태코드
+ HTTP 상태코드란 무엇이고, 상태코드중 중요한 것이 무엇이고 어떤 역할인지 찾아볼 것

- 필수 HTTP 상태코드
+ 200: OK(성공)

+ 201: Created(생성됨)

+ 400: Bad Request(잘못된 요청)

+ 401: Unauthorized(실패)

+ 403: Forbidden(금지됨, 401과는 다른 용도)

+ 404: Not Found(리소스 또는 경로를 찾을 수 없음)

+ 408: Time out (요청 시간 초과)

+ 422: Unprocessable Content(데이터 타입이 잘못됨)

+ 500: Internal Server Error(서버 내부 오류)

```
